# 智慧树自动互动
智慧树自动刷互动分
## 如何使用
1.登录智慧树<br/>
2.进入我的学堂按下f12<br/>
3.在网络里面找到getStudentStudyInfo（其他的get也行），复制里面的cookie<br/>
4.同目录下创建cookie.txt将cookie复制进去<br/>
5.运行py<br/>

## 注意事项<br/>
1.请事先pip requests,json<br/>
2.time.sleep(3)只能变大不能变小，防封号<br/>
